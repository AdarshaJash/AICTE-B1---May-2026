# AI-Powered Study Buddy

## Overview
AI-Powered Study Buddy is a Streamlit-based educational assistant that helps students learn efficiently using Artificial Intelligence.

## Features

* Concept Explainer
* Notes Summarizer
* Quiz Generator
* Study Planner

## Technologies Used

* Python
* Streamlit
* Google Gemini API
* PyPDF2
* Python Dotenv

## How to Run
1. Install dependencies
pip install -r requirements.txt

2. Run the application
streamlit run app.py
